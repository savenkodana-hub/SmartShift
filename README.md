# 📱 SmartShift - מערכת חכמה לניהול משמרות ושכר

SmartShift היא אפליקציית אנדרואיד מקיפה המייעלת את תהליך ניהול העובדים בעסקים. האפליקציה מחברת בין מנהלים לעובדים בזמן אמת, ומספקת פתרון מקצה לקצה: החל מהגשת זמינות ושיבוץ משמרות, דרך שעון נוכחות חי, ועד לחישוב שכר אוטומטי והפקת תלושי שכר (PDF).

## ✨ תכונות מרכזיות

### 👨‍💼 צד מנהל (Manager)
* **ניהול מצבת עובדים וקביעת שכר:** צפייה ברשימת העובדים המלאה של העסק וקביעת שכר שעתי (תעריף) לכל עובד בנפרד.
* **בניית סידור עבודה חכם:** שיבוץ עובדים למשמרות השבוע. המערכת כוללת התראות חכמות - עובד שלא הגיש זמינות למשמרת מסוימת יופיע באדום (❌) כדי למנוע טעויות שיבוץ.
* **סטטוס עובדים בזמן אמת (Live Status):** מסך ייעודי המאפשר למנהל לראות בדיוק מי מהעובדים נמצא כרגע במשמרת (עשה כניסה) ומי לא.
* **דוחות שכר ומשכורות:** מעקב שוטף אחרי שעות העבודה של כלל העובדים וצפייה בדוחות שכר חודשיים מרוכזים.

### 👷‍♂️ צד עובד (Employee)
* **שעון נוכחות (כניסה/יציאה):** החתמת כרטיס וירטואלית בתחילת ובסיום משמרת.
* **חישוב שכר אוטומטי:** בעת "יציאה ממשמרת", המערכת מחשבת אוטומטית את השכר לאותו יום (לפי השכר השעתי שקבע המנהל), מציגה את הסכום לעובד ומעדכנת את דוחות השכר של העסק.
* **הגשת משמרות וצפייה בסידור:** ממשק נוח לסימון זמינות (בוקר/צהריים/לילה) לשבוע הבא, וצפייה במשמרות שאליהן העובד שובץ.
* **שוק החלפות משמרות (Shift Swap):** עובד שלא יכול להגיע למשמרת יכול ללחוץ עליה **לחיצה ארוכה** בסידור העבודה ולהציע אותה להחלפה. המשמרת תופיע ב"שוק ההחלפות", ושאר העובדים יוכלו לקחת אותה במקומו.
* **סנכרון ליומן (Google Calendar):** ייצוא אוטומטי של כל המשמרות העתידיות ישירות ליומן האישי במכשיר בלחיצת כפתור.
* **הורדת תלוש שכר (PDF):** הפקת דוח שעות חודשי והורדתו למכשיר כקובץ PDF מסודר.

## 📖 מדריך משתמש (User Guide)

### איך להשתמש באפליקציה כמנהל?
1. **הרשמה ויצירת עסק:** בעת ההרשמה, בחר בתפקיד "מנהל". קבע **קוד עסק** ייחודי משלך (Business Code) ומסור אותו לעובדים שלך כדי שיוכלו להצטרף.
2. **הגדרת שכר לעובדים:** לאחר שהעובדים נרשמו, היכנס למסך ניהול העובדים והגדר לכל עובד את התעריף השעתי שלו.
3. **שיבוץ משמרות:** היכנס למסך "סידור עבודה", בחר את השבוע הרצוי ולחץ על משבצות ריקות כדי לשבץ עובדים. (שים לב לאזהרות האדומות לגבי עובדים שאינם זמינים).
4. **מעקב שוטף:** במהלך היום, תוכל להיכנס למסך "זמן אמת" (Live Status) כדי לראות מי עובד כרגע, ובסוף החודש תוכל לעיין בדוחות השכר המרוכזים.

### איך להשתמש באפליקציה כעובד?
1. **הרשמה והצטרפות לעסק:** בעת ההרשמה, בחר בתפקיד "עובד". המערכת תבקש ממך להזין את **קוד העסק** שקיבלת מהמנהל שלך. הזן אותו כדי להסתנכרן עם העסק הנכון.
2. **ניהול משמרות קדימה:** * היכנס ל"הגשת משמרות" וסמן מתי אתה יכול לעבוד בשבוע הבא.
   * ב"משמרות שלי" תוכל לראות את סידור העבודה שלך, ולסנכרן אותו ליומן גוגל בלחיצת כפתור.
3. **החלפת משמרת:** נתקעת? כנס לסידור העבודה, לחץ **לחיצה ארוכה** על המשמרת שלך ובקש להחליף אותה. עובד אחר יוכל להיכנס ל"שוק ההחלפות" ולקחת אותה במקומך.
4. **שעון נוכחות ושכר:**
   * בתחילת משמרת, לחץ על **"כניסה למשמרת"**.
   * בסיום המשמרת, לחץ על **"יציאה"**. המערכת תחשב מיד כמה הרווחת היום ותעדכן את הנתונים.
   * בסוף החודש, תוכל להוריד את ריכוז השעות והשכר שלך כקובץ **PDF**.

## 🛠️ טכנולוגיות
* **שפת פיתוח:** Java
* **סביבת עבודה:** Android Studio
* **מסד נתונים ואימות:** Firebase Realtime Database & Firebase Authentication
* **רכיבי מערכת:** Android Calendar Provider API (לסנכרון יומן), PDF Document API (להפקת תלושים).
